Compilation and Installation:
I am using this toolchain: https://github.com/is1200-example-projects/mcb32tools
make
make install TTYDEV=/dev/ttyS2
(ttyS# # -> might be different than 2)

Main Menu Controls:
BTN4: Select
BTN3: Go Down
BTN2: Go Up

Game Controls (Snake / Space Invaders):
BTN4: Go Up / Shoot
BTN3: Go Down / Special Ability (Not always active, you have to have sufficient combo)
BTN2: Go Left
BTN1: Go Right
